Projetos

Todos os projetos dessa seção foram criados em diferentes partes do meu aprendizado, alguns estão mais simples justamente porque foi o que eu consegui elaborar enquanto ainda estava no inicio dos estudos sobre programação.

Nenhum projeto foi copiado da internet, criei todos inteiramente sozinho pegando apenas referencias de projetos básicos em outros perfis.

Calculadora

Uma calculadora que realiza as operações básicas da matemática (Soma, Subtração, Multiplicação e Divisão) e com recurso para alterar o estilo com outras cores: Padrão (Azul), Vermelho, Verde, Roxo, ou Alto Contraste

Login - Cadastro - Senha

Criei 3 telas para praticar o uso de validações básicas de formulário, onde é possível realizar o cadastro seguindo determinadas condições, alterar a senha cadastrada de acordo com determinados parametros e realizar o login.

Lista de tarefas

Um sistema CRUD bem simples para criar, visualizar, alterar e deletar tarefas de uma lista. É uma tela simples, porem, existe uma programação com Javascript que permite o usuário ter maior controle sobre as atividades inseridas.

Pagina Talkd V1

Essa página foi criada como um "teste" para substituir a página que era utilizada pela empresa Talkd Soluções em Atendimento, local onde eu trabalhei como Auxiliar de projetos até pouco tempo atrás.

Pagina Talkd V2

Essa é uma versão mais elaborada da página criada para a Talkd Soluções em Atendimento, após criar a página de testes e ver que o pessoal aceitou a idéia eu decidi me empenhar em criar uma página com mais recursos.
